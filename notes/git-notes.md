## Basic commands:

```
$ git init // *Create an empty Git repository or reinitialize an existing one*
$ git add . //
$ git add someFile.js //
$ git commit -m ¨some message¨ //
$ git commit -a -m ¨some message¨ //
$ git push -u origin master //
```

```
$ git fetch
$ git merge
$ git pull
$ git diff HEAD
$ git diff
```
